hexo 搭建github page 静态博客

学习 
hexo 搭建github page静态博客

1、安装nodejs（这个自行百度）

2、npm install -g hexo-cli 安装hexo的cli模块 g代表全局相当于环境变量

3、去一个目录创建一个英文文件夹 用dos进入该目录下

4、hexo init ————folder是hexo博客项目的文件名字随意取

5、cd 进入4操作产生的文件中

6、修改 目录下的_config.yml 中的 deploy部分配置

如下：
    deploy:
          type: git   
          repo: git@github.com:19y/yyy.git   //写你的git项目的上传地址
          branch: master

除了将repo改为你的地址以外其他不用动

7、 6步骤的时候你可能还没有git仓库–所以去注册一个。。自行百度很简单，跟注册qq一样

8、 npm install hexo-deployer-git –save 安装hexo 自动push git需要的模块

9、至此基本都完事了就差部署到github 了

10、cd 使用 hexo new post “文章名字，随便取”

11、 完事后你会在/source/_posts 发现一个以你刚刚文章名字命名的文件

12、上边的东西不用动，直接在下边写你要写的文章即可， 文章格式可以在hexo上查看api

13、写完文章了 cd 使用 hexo g

14、 hexo s 是在本地运行 –这里我们直接使用hexo d 将项目部署到git上

15、点开你的仓库查看上方有个Setting 点击它 进入新的页面后一直往下滑动发现

17、当然你会发现主题不好看，你可以去官网上方点击主题，去git 上把喜欢的主题pull来，然后将pull的主题文件放在
/themes文件夹中然后修改/_config.yml中的theme： 修改为你放在themes下的文件夹的名字，其余的配置查看说明

18、我觉的大家没必要仔仔细细的把官网的api过个遍，只要能使用常用的功能就可以，比如更换主题，发布文章就行。
毕竟结果至上。
