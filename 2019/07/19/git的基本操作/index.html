git的基本操作

学习

git 工程的生成
git init

git 添加缓存
git add ‘文件名或一些特定表达式如：*代表所有’

git 添加的本地厂
git commit -m ‘ annotation’

git 添加一个源 ，用来push 代码用的
git remote add origin[名字随便取] url[链接]

git 删除一个源
git remote remove origin[源名字]

git push 提交代码
git push -u origin master //第一此加u why 看笔记
